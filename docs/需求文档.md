# 选课管理系统 - 需求文档 (PRD)

## 1. 项目概述

- **项目名称**: 选课管理系统 (Course Selection System)
- **项目类型**: Web 前后端分离应用
- **核心功能**: 学生选课、教师管理课程、系统管理员管理
- **目标用户**: 学生、教师、管理员

## 2. 功能需求

### 2.1 用户角色

| 角色 | 功能 |
|------|------|
| **学生** | 浏览课程、选课、退课、查看已选课程 |
| **教师** | 创建课程、修改课程信息、查看选课学生 |
| **管理员** | 用户管理、课程管理、系统配置 |

### 2.2 核心功能

#### F1: 用户管理
- 用户注册/登录 (JWT 认证)
- 角色分配 (学生/教师/管理员)
- 个人信息查看与修改

#### F2: 课程管理
- 教师创建课程 (课程名称、简介、学分、容量)
- 课程列表展示 (分页、筛选)
- 课程详情查看

#### F3: 选课功能
- 学生选课 (检查容量、冲突)
- 学生退课
- 选课时间限制

#### F4: 数据展示
- 学生：已选课程列表
- 教师：所授课程及选课学生列表
- 管理员：全量数据统计

## 3. 非功能需求

- **性能**: 页面加载 < 2s，支持 1000+ 并发
- **安全**: JWT 认证、密码加密
- **易用**: 界面简洁、操作流畅

## 4. 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue2 + Vite + Axios |
| 后端 | Spring Boot 3 + MyBatis Plus |
| 数据库 | MySQL 8 |
| 部署 | Docker + Nginx |

## 5. 风险评估

- 中等风险：涉及用户认证和权限管理
- 需注意选课并发控制
